# Criminal-Face-Detection
🚔 Criminal Face Detection and Identification System using Java, OpenCV & Machine Learning

I'm excited to share my project:

🔍 Criminal Face Detection and Identification System

This project is designed to assist law enforcement agencies by automatically detecting and recognizing criminals using facial recognition technology. The application provides an easy-to-use interface for registering criminals, training face data, and identifying suspects from images or live video. <br>


🚀 Features<br>
👤 Criminal Registration with Photo<br>
📷 Face Detection using Haar Cascade Classifier<br>
🧠 Face Recognition using LBPH Algorithm<br>
🎥 Real-Time Video Surveillance<br>
🖼️ Image-Based Criminal Detection<br>
💾 MySQL Database Integration<br>
📄 Automatic Report Generation<br>
🖥️ Java Swing GUI<br>
