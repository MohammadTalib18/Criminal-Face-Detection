# Criminal-Face-Detection
🚔 Criminal Face Detection and Identification System using Java, OpenCV & Machine Learning

I'm excited to share my project:

🔍 Criminal Face Detection and Identification System

This project is designed to assist law enforcement agencies by automatically detecting and recognizing criminals using facial recognition technology. The application provides an easy-to-use interface for registering criminals, training face data, and identifying suspects from images or live video.


🚀 Features
👤 Criminal Registration with Photo
📷 Face Detection using Haar Cascade Classifier
🧠 Face Recognition using LBPH Algorithm
🎥 Real-Time Video Surveillance
🖼️ Image-Based Criminal Detection
💾 MySQL Database Integration
📄 Automatic Report Generation
🖥️ Java Swing GUI
